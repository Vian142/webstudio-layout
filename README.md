# Верстка макета Веб-студии

##### Установка пакетов
```sh
$ npm install
```

##### Запуск Gulp - watch
```sh
$ gulp
```

##### Сборка проекта
```sh
$ gulp build
```

В проекте используется:
 - [Bootstrap](http://getbootstrap.com/)
 - [Normalize-css](https://necolas.github.io/normalize.css/)
 - [FontAwesome.io](http://fontawesome.io/)
